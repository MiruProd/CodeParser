# CodeParser

CodeParser — это компактное настольное приложение для Windows, предназначенное для быстрой подготовки и упаковки исходного кода ваших проектов в единый текстовый файл (контекст) для последующей передачи в нейросети (ChatGPT, Claude, Gemini и другие LLM).

Приложение избавляет от ручного копирования файлов и помогает передавать в чат-бот только нужную кодовую базу, отсекая бинарные данные, тяжелые лок-файлы и системный мусор.

---

## 🚀 Как начать работу (Без установки)

Вам не нужно устанавливать Python или настраивать виртуальные окружения для запуска программы:
1. Перейдите в раздел **Releases** (выпуски) на панели вашего репозитория справа.
2. Скачайте готовый исполняемый файл `CodeParser.exe`.
3. Запустите его на вашем компьютере с Windows. Программа работает в портативном режиме без установки.

---

## ⚙️ Как это работает

Приложение объединяет сканирование директорий проекта с удобной визуальной настройкой экспорта:
1. **Выбор папки проекта:** Вы указываете путь к каталогу вашей программы.
2. **Автоматическая чистка:** На основе заданных параметров фильтрации программа сама скрывает из дерева каталоги виртуального окружения (`venv`), кэш, лок-файлы (`package-lock.json`, `yarn.lock`, `go.sum` и т.д.) и бинарные форматы (картинки, архивы).
3. **Выбор файлов вручную:** В интерактивном дереве папок и файлов вы отмечаете галочками только то, что требуется для текущего запроса к нейросети.
4. **Копирование или сохранение:** В один клик вы копируете сформированный структурированный текст в буфер обмена или сохраняете его в файл `code_context.txt`.

---

## ✨ Что сделано и реализовано в приложении

*   **Портативность:** Работает напрямую из одного EXE-файла, без скрытых зависимостей.
*   **Интерактивное дерево каталогов:** Реализовано удобное каскадное управление чекбоксами (выделение или снятие выделения с родительской папки автоматически применяется ко всем вложенным файлам и папкам).
*   **Поддержка правил `.gitignore`:** Программа умеет считывать локальный файл `.gitignore` в вашем проекте и использовать его правила для фильтрации структуры.
*   **Калькулятор объема и токенов:** В реальном времени рассчитывается суммарный размер выбранных вами файлов в килобайтах, а также выводится ориентировочное количество токенов для LLM.
*   **Готовые шаблоны (Пресеты):** Быстрая фильтрация по расширениям для популярных стеков разработки (Python, Go/Wails/Svelte, Web/Frontend, Rust, C/C++, Java/Kotlin) выбирается в один клик из выпадающего списка.
*   **Две темы оформления:** Встроен переключатель между Темной темой (в стиле графического редактора VS Code) и Светлой темой.

## 📂 Пример формата выходных данных

В результате выгрузки формируется структурированный текст, который легко и без ошибок распознается любыми моделями искусственного интеллекта:

```text
=== ПОЛНАЯ СТРУКТУРА ПРОЕКТА (БЕЗ СИСТЕМНОГО МУСОРА) ===

MyProject/
├── src/
│   ├── core/
│   │   └── parser.py
│   └── main.py
└── README.md

================================================================================

=== СОДЕРЖИМОЕ КЛЮЧЕВЫХ ФАЙЛОВ КОДА ===

<file path="src/main.py">
# Содержимое файла main.py...
</file>

<file path="src/core/parser.py">
# Содержимое файла parser.py...
</file>
```

## 🛠 Запуск из исходного кода (Для разработчиков)

Если вы хотите запустить проект в режиме разработки напрямую из кода:
1. Убедитесь, что на компьютере установлен Python версии 3.10 или выше.
2. Установите зависимость (библиотеку PyQt6):
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите утилиту:
   ```bash
   python src/main.py
   ```
